Går det an? kopiere mappe og innhold med én gang til Github

Nå har jeg laget en mappe (testGH) med denne HTML fil (test.html) inni og henvisning til en bilde (bilde.jpg) som står i mappe bilde, selvfølgelig).

Da skal jeg prøve å kopiere til GitHub (wish me luck!).

Yippie! Det går bra. Hvordan?

  1. Logg inn i GitHub

    Klikk på "Upload files" (bilde nedenfor)

  2. dra mappe inn i repository (bilde nedenfor)

  3. Fyll inn hva du har gjort og trykk "Commit changes" (grønn knappe) (bilde nedenfor)

Bra gjort!
